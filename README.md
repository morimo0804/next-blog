# リコレブック

Next.js × Tailwind CSS × microCMS を使って制作したブログサイトです。
1 年次に制作したブログをベースに、使用技術を変えて再度制作しました。

## 使用技術

- Next.js
- React / TypeScript
- Tailwind CSS
- microCMS（ヘッドレス CMS）
- react-slick（カルーセル）

## 機能・特徴

- ブログ記事の投稿・表示（microCMS 連携）
- カテゴリー分け、記事検索、ページネーション
- ピックアップ記事のスライダー表示
- Tailwind CSS によるデザイン再現
